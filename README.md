### Why create another virtualenv?
```
I just wanted shorter commands.

Because who wants to have to type this all the time?

$ virtualenv venv 
```

## Install

Install with pip

```bash
$ sudo pip install virtualenvy
```
## Run

create a virtualenv

```bash
$ virtualenvy
```
